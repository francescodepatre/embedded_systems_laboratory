Thingy53 DFU firmware

This archive contains the DFU package with firmware for Nordic Semiconductor Thingy53.
To flash this firmware, you need a Thingy53 with pre-flashed mcuboot. You can use a serial or BLE connection for flashing purposes.

See instructions on the Nordic documentation page:
https://developer.nordicsemi.com/nRF_Connect_SDK/doc/latest/nrf/ug_thingy53_gs.html#updating-through-usb

See Getting Started Guide for more details:
https://docs.edgeimpulse.com/docs/development-platforms/officially-supported-mcu-targets/nordic-semi-thingy53
